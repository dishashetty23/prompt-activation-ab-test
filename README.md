# Edge Copilot A/B Experiment

An end-to-end A/B test design exploring whether showing a contextual Copilot prompt early in a new user's experience increases the likelihood of them setting Edge as their default browser.

---

## What this covers

- Experiment hypothesis, metric structure, and randomization design
- Power analysis with sensitivity table across MDE assumptions
- Simulated binary outcomes and statistical test (two-proportion z-test)
- 95% confidence interval with ship / hold / no-ship decision logic
- Search query guardrail check
- Notes on assignment balance, novelty effect, and causal chain
